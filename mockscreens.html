Login page and registration page

The login and registration page are the first screens a user would see on first installation of the application.Both screens asks for the basic requirement a user would use for either login or registration.The login and registration page also comes with an alternative way to enable user login and registration by using Oauth.This way a user can allow access to either there google or facebook account for authentication.

Home page

Performing a blood sugar recording task

As a user logins into the application and is presented with the home screen he clicks on the blood sugar button and the next screen opens on the right which is the blood sugar dashboard where all blood sugar readings entered by peter will be recorded.As he clicks on the plus icon he is presented with the screen to the right of the dashboard to add the sugar reading gotten from the Glucose meter he used for measurement.When the user is done with adding the reading he is directed back to the dashboard screen to view the addition that he just made.
